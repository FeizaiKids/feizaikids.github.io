##### install node.js
https://nodejs.org/en/download/

```
node --version
npm -v
```

##### install vue cli
```
npm i -g @vue/cli @vue/cli-service-global //cli-service-global: zero-configuration
```
##### save App.vue in a folder
```
<template>
    <h1>Hello World</h1>
</template>
```

##### start server
switch to the folder
```
vue server -o
```
